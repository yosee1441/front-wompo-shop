# WOMPO SHOP

## Description
The Wompo Shop project is built on Node V18.18.0

## Installation
```bash
$ npm install
```

## Running the app
```bash
# develop
$ npm run start

# production
$ npm run build
```

## Test
```bash
# unit tests
$ npm run test


# test coverage
$ npm run test:cov
```
